# AdminX – Admin Dashboard UI

A modern Admin Dashboard built with React, inspired by the Figma design.

## Features
- Sidebar with navigation and user profile
- Top navbar with search, notifications, and profile
- Dashboard stat cards with mini bar charts
- Revenue bar chart panel
- Traffic sources with progress bars
- Employee table with filter pills, search, and status badges

## Tech Stack
- React 18 (functional components + hooks)
- CSS Modules (component-scoped stylesheets)
- Inter font via Google Fonts

## Getting Started
```bash
cd DashboardProject
npm install
npm start
```
