# PhonePe React UI

A PhonePe-style mobile payment UI built with React.

## Features
- Header with profile avatar and greeting
- Wallet balance card with show/hide toggle
- Quick actions grid (8 tiles)
- Recent transactions with See All toggle
- Bottom navigation bar with active state

## Getting Started
```bash
npm install
npm start
```
